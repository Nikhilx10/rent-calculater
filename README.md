#  Rent Calculator

A simple Python project that helps flatmates split their monthly expenses without doing the calculations manually.

Whether it's rent, food expenses, or electricity bills, this calculator quickly tells each person how much they need to pay.
##  Features
- Add flat rent
- Add food expenses
- Calculate electricity charges
- Split the total amount equally among all roommates
- Simple and beginner-friendly Python project

model link-https://rent-calculater-cc92srcsvdvzce2xxpdigl.streamlit.app/

##  Built With
- Python 3

##  How to Run

1. Download or clone this repository:

```bash
git clone https://github.com/YOUR_USERNAME/rent-calculator.git
```

2. Move to the project folder:

```bash
cd rent-calculator
```

3. Run the program:

```bash
python rental.py
``
```
